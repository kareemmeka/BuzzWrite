Pseudocode:

EVERY FRAME:
  1. Get laser tip position (from SimpleHandPointer)
  2. Get letter collider bounds
  3. Find closest point on letter to laser tip
  4. Calculate distance
  
  IF distance < detectionRadius (0.08 units):
    state = ON_PATH
    Set laser color to GREEN
    Stop haptic vibration
  ELSE:
    state = OFF_PATH
    Set laser color to RED
    Start haptic vibration
  END IF
